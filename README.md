

### Getting Started

First, run the development server:

```bash
npm install 
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### `Technologies`

* Next.js
* Express
* TypeScript
* Material-UI
* Prisma

### `Functions`

* Add new task
* Sort tasks
* Delete task
* Edit status task
* After save log all task in console
